How to Use — 使い方

1. ダイビングを記録する
  1. ホームの 記録する →「ダイビングを記録」を開く
  2. その日の共通情報（日付・場所・通算本数・ウェイト・バディ）を入力。前回の値が自動で入っています
  3. 1本ごとにポイント・費用・見れた魚・コメントを入力して保存。「+ もう1本追加」で連続入力できます

潜水時間・最大水深・平均水深は任意入力です。ダイブコンピューターのCSVを後で取り込む人は空欄のままでOK。ダイコンを使わない人はここに手入力してください。

💡 天気・波高・水温は場所と日付から自動で取得されます（取得できない場合は空欄になります）
2. 魚を図鑑に登録する

ダイブ記録の「見れた魚」欄で名前を入力すると、自分の図鑑から検索されます。まだ図鑑にない魚は「新規追加」を選ぶと、保存時にまとめて図鑑登録できます。

あとから追加したい場合は「🐠 見れた魚を追加」から。写真の登録・変更は My図鑑の魚カードを開いて 📷 写真を変更 からできます。

3. 写真・動画・総評を追加する
  • 写真 — 「写真をアップロード」から。自動で縮小されるので大きい写真もそのままでOK
  • YouTube動画 — 「🎬 YouTube動画を追加」でダイブを選んでURLを貼るだけ
  • その日の総評 — 「📝 その日の総評を書く」またはセッションページの ✏️ から
4. ダイコンCSVで自動入力する
  1. 先に「ダイビングを記録」で1本ずつのログを作っておく（潜水データは空欄で）
  2. ダイブコンピューターからCSVをエクスポート
  3. 「📥 ダイコンCSVで自動入力」でファイルを選ぶと、同じ日付のログと照合されます
  4. 照合内容を確認してから更新実行。深度グラフも自動で付きます
5. あとから編集する
  • 総評・ダイブのコメント — セッションページの文章の右下にある ✏️ をタップ
  • 図鑑の内容 — My図鑑で魚カードを開き、一番下の ✎ 登録内容を編集する をタップ
6. ログブックをSNSで共有する

ホームの 公開リンクをコピー を押すと、閲覧専用の公開URLがコピーされます。このURLを知っている人は誰でも（ログイン不要で）あなたのログブックを見られますが、編集や記録はできません。

⚠️ 公開したくない記録がある場合はURLの共有範囲に注意してください
7. データについて

あなたのログはあなたのアカウントでログインした人にしか見えません。他のメンバーのログを見ることはできず、逆も同じです（公開リンクを共有した場合を除く）。
