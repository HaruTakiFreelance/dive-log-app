How to Use — 使い方

1. ダイビングを記録する
  1. ホームの 記録する →「ダイビングを記録」を開く
  2. その日の共通情報（日付・場所・通算本数・ウェイト・バディ）を入力。前回の値が自動で入っています
  3. 1本ごとにポイント・費用・見れた魚・コメントを入力して保存。「+ もう1本追加」で連続入力できます

潜水時間・最大水深・平均水深は任意入力です。ダイブコンピューターのCSVを後で取り込む人は空欄のままでOK。ダイコンを使わない人はここに手入力してください。

💡 天気・波高・水温は場所と日付から自動で取得されます（取得できない場合は空欄になります）
2. 魚を図鑑に登録する

ダイブ記録の「見れた魚」欄で名前を入力すると、自分の図鑑とマスター図鑑（約1300種）の両方から検索されます。「📖 図鑑から」と付いた候補を選ぶと、英名・学名・科などが自動で入った状態で自分の図鑑に登録されます。マスターにない魚は「新規追加」から手入力で登録できます。

あとから追加したい場合は「🐠 見れた魚を追加」から。写真の登録・変更は My図鑑の魚カードを開いて 📷 写真を変更 からできます。

My図鑑の「すべて」タブでは全1264種が科ごとに並び、まだ見ていない魚は黒いシルエットで表示されます。科の見出し（例: ハゼ科 12/132）をタップすると中身が開き、上の検索ボックスから名前で直接探すこともできます。ページ上部には図鑑完成率のバーが表示されるので、コンプリートを目指しましょう。

図鑑の内容を編集しても変わるのは自分の図鑑だけで、他のメンバーには影響しません。

📷 未発見種の参考写真は iNaturalist のクリエイティブ・コモンズ画像を利用しています（詳細画面にクレジット表記）
3. 写真・動画・総評を追加する
  • その日の写真 — 「写真をアップロード」から。セッションページの上部にまとめて表示されます。自動で縮小されるので大きい写真もそのままでOK
  • ダイブごとの写真 — 「📷 ダイブごとに写真を追加」でダイブを選んでアップロード。そのダイブのカード内（動画の上）に表示されます
  • YouTube動画 — 「🎬 YouTube動画を追加」でダイブを選んでURLを貼るだけ
  • その日の総評 — 「📝 その日の総評を書く」またはセッションページの ✏️ から
4. ダイコンCSVで自動入力する
  1. 先に「ダイビングを記録」で1本ずつのログを作っておく（潜水データは空欄で）
  2. ダイブコンピューターからCSVをエクスポート
  3. 「📥 ダイコンCSVで自動入力」でファイルを選ぶと、同じ日付のログと照合されます
  4. 照合内容を確認してから更新実行。深度グラフも自動で付きます
5. あとから編集する
  • 総評・ダイブのコメント — セッションページの文章の右下にある ✏️ をタップ
  • 図鑑の内容 — My図鑑で魚カードを開き、一番下の ✎ 登録内容を編集する をタップ
6. テーマを自分好みにする

ホームの テーマ から、ログブックの見た目を自由に変えられます。

  • カラー — 深海 / 墨 / 夕凪 / 南国 の4種類
  • 背景写真 — アップロード済みの写真から選ぶか、📁 アップロード でスマホの写真フォルダから直接設定できます
  • フォント — 明朝・ゴシック・丸ゴシックなど5種類

タップするとその場でプレビューされ、👁 背景を確認 で全体を確認できます。保存して閉じる で確定。設定はアカウントに保存され、どの端末でも同じ見た目になります。公開リンクを見た人にも同じテーマで表示されます。

💡 「保存に失敗: ログインの有効期限が…」と出た場合は、案内に従ってログインし直せばOKです
7. ログブックをSNSで共有する

ホームの 公開リンクを取得 を押すと、閲覧専用の公開URLがコピーされます。このURLを知っている人は誰でも（ログイン不要で）あなたのログブックを見られますが、編集や記録はできません。あなたが設定したテーマ（背景写真・カラー・フォント）もそのまま反映されます。

⚠️ 公開したくない記録がある場合はURLの共有範囲に注意してください
8. スマホでアプリのように使う

Safariでこのサイトを開き、共有ボタン → ホーム画面に追加 を選ぶと、アイコン付きのアプリとして使えるようになります（アドレスバーも消えて全画面表示になります）。

9. アカウントとデータについて

初めての方は、ログイン画面の Googleでログイン を押すだけでアカウントが作成されます。

あなたのログはあなたのアカウントでログインした人にしか見えません。他のメンバーのログを見ることはできず、逆も同じです（公開リンクを共有した場合を除く）。
